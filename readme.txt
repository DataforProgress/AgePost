This is the source code for the models behind the dfp post on age and politics https://www.dataforprogress.org/blog/6/14/progressives-control-the-future
You have to first download the ANES cumulative data file and place it in the \Data folder.
This requires that you create an account with ANES but the data is free.
The anes.py script preprocesses the raw data file and prepares it for use with runGP.py, which runs the gaussian process models
Questions can be directed to colin@dataforprogress.org
